# url-shortener-go

## How To Run The Program
    
  For Client (React): 
  
    cd client
    npm run dev

  For Server Side:
  
    cd server
    go run main.go

➔ The user may upload their own personalised shorts.

➔ An anonymous user can only add urls, but a registered user can see, edit, add, and remove urls.

➔ TECH STACK : Golang, React.js, MongoDB
